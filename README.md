## Getting Started

First, clone repository
```bash
git clone https://github.com/AlbertoAndrade10/GEA-Official
```

Intall dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
